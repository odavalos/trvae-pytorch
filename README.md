# trvae-pytorch

A repackaging of [trvaep](https://github.com/theislab/trvaep) due to original having possible circular import.


#### Install Package Locally
Make sure to be in the same directory as `setup.py`. Then, using `pip`, run:

````bash
pip install -e .
````
